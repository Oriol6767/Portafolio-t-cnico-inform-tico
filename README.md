## 🛠️ Soporte Técnico y Mantenimiento de Equipos

En esta sección incluyo prácticas y conocimientos relacionados con el soporte técnico, el mantenimiento de equipos y la resolución de incidencias, alineados con las tareas habituales de un técnico informático.

### 🔧 Mantenimiento de equipos
- Instalación y actualización de sistemas operativos.
- Instalación de drivers y software básico.
- Revisión de hardware y diagnóstico de fallos comunes.
- Limpieza interna y mantenimiento preventivo de equipos.
- Sustitución de componentes básicos (RAM, discos, fuentes, etc.).

### 🖥️ Administración básica de sistemas
- Gestión de usuarios y permisos en Windows y Linux.
- Configuración inicial de equipos.
- Actualización de sistemas y antivirus.
- Monitorización básica del estado del sistema.

### 🌐 Redes y conectividad (Cisco)
- Conceptos básicos de redes: IP, máscara, puerta de enlace, DNS.
- Configuración inicial de switches y routers (teórica).
- Comprobación de conectividad y resolución de incidencias.
- Prácticas con VLANs básicas y direccionamiento.

### 🖧 Soporte remoto
- Uso de herramientas de asistencia remota.
- Resolución de incidencias de software y red a distancia.
- Guías de diagnóstico rápido para usuarios.

### 🧰 Virtualización
- Creación de máquinas virtuales para pruebas.
- Instalación de sistemas en entornos virtualizados.
- Uso de virtualización para practicar configuraciones sin riesgo.

### 🎯 Objetivo
Desarrollar habilidades prácticas en soporte técnico, mantenimiento de equipos, redes y administración básica de sistemas para aplicarlas en entornos reales
